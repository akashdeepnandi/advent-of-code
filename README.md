## Commands:

JavaScript:

```bash
cd yYY/dDD/javascript
node index.js input
```

Golang:

```bash
cd yYY/dDD/golang
go run . -f input
```

Python:

```bash
cd yYY/dDD/python
python solve.py input
```

Rust:

```bash
cd yYY/dDD/rust
cargo run input
```

CPP:

```bash
cd yYY/dDD/cpp
g++ main.cpp helper.cpp && ./a.out input
```
